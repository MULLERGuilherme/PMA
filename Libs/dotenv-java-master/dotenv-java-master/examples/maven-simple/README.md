# dotenv-java-example

Simple `dotenv-java` example using maven.

## Clone

```shell
git clone 
```

## Compile

```shell
mvn compile
```

## Run

```shell
mvn exec:java -Dexec.mainClass="io.github.cdimascio.examples.dotenv.Main"
```

The program outputs the value of the environment var `MY_ENV`. In this case it's, `MY_VALUE`
